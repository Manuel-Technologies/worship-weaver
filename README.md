# AmboPro

**Everything in sync.**

---

## Welcome to AmboPro

AmboPro is a modern, AI-powered church presentation and projection software designed to simplify worship services.

It helps churches automatically display scriptures, song lyrics, and sermon slides in real-time, allowing ministers and media teams to focus on what truly matters — the message.

---

## ⚡ What Makes AmboPro Different

- Real-time scripture detection  
- Automatic slide generation  
- Seamless HDMI projection to TV and projectors  
- Smart service flow management  
- Clean, distraction-free presentation  

---

## 🚀 Core Features

### 🎙️ Live Audio Recognition
AmboPro listens to sermons and detects scripture references instantly.

### 📖 Automatic Scripture Display
Relevant Bible verses are displayed automatically without manual input.

### 🎵 Worship Integration
Song lyrics are detected and projected in sync with worship.

### 🖥️ HDMI Output
Direct projection to external screens, TVs, and projectors.

### ⚙️ Smart Control
Manual override available when needed, with full preview control.
